文章标题：Adversarial Camouflage: Hiding Physical-World Attacks With Natural Styles

代码：https://github.com/RjDuan/AdvCam-Hide-Adv-with-Natural-Styles

1.摘要与技术路线

实现了针对分类器的伪装对抗样本生成，进行了数字攻击和物理攻击实验验证。

这张技术路线图有4个关键点值得注意：

  • Customization部分：注意有个遮罩，并且有风格图像，这个是否需要针对具体图像进行人工设计？有无更好的解决方案？
  • Physical Adaptation部分：进行了多样的图像变换，注意将其集成到优化循环中的方式。
  • Feature Extrator：具体是如何提取的特征，如何得到的Style Loss？
  • 扰动是如何优化得到的，是否直接修改像素值，还是有另外一个扰动生成网络？（推测扰动不是图像间通用的）

2.关键方法

2.1遮罩的使用

遮罩这个看起来不起眼的技术，感觉起到了一个关键作用，使得不同任务之间不会产生直接的冲突。分析模型输入，该模型共有两个遮罩：content mask 和 style mask，会在计算 style loss和完成物理条件的变换时用到。

2.2风格化技术：构造 Style Loss

Gram矩阵

Gram矩阵是两两向量的内积组成，所以Gram矩阵可以反映出该组向量中各个向量之间的某种关系。有点类似协方差矩阵，不过这里的对象是向量不是随机变量。

风格一致的度量

使用Gram矩阵进行风格的度量，将两张图片分别送入CNN网络，将卷积网络中间的某几层特征向量取出，分别计算两张图片的这些特征向量的Gram矩阵，然后计算两个Gram矩阵的相似度，越小则风格越接近。
$$
D_s = \sum_{l\in S_l} ||G(F_l(x^s))-G(F_l(x’))||_2^2
$$
本模型中计算Style Loss具体使用的特征向量为：vgg.conv1_1, vgg.conv2_1, vgg.conv3_1, vgg.conv4_1, vgg.conv5_1。

2.3内容损失：Content Loss

使用 vgg.conv4_2作为内容损失计算的特征值。没有使用mask。

def content_loss(const_layer, var_layer, weight):
    return tf.reduce_mean(tf.squared_difference(const_layer, var_layer)) * weight

2.4对抗损失：Adversarial Loss

分为有目标损失和无目标损失。

  • 有目标损失
def targeted_attack_loss(pred, orig_pred, target, weight):
    """
    Note:
        balance can be adjusted by user for better results, range in [2,5] is recommended
    Arguments:
        pred {logits} -- Logits output by threat model (input: adv) 
        orig_pred {int} --  Original (correct) prediction by threat model
        target {int} -- Target lable assigned by user, range in [0,999]
        weight {float32} -- Attack weight assigned by user (args.attack_weight)
    
    Returns:
        [type] -- [description]
    """
    balance = 5
    orig_pred = np.eye(1000)[orig_pred]
    target = np.eye(1000)[target]
    loss1 = -1 * tf.nn.softmax_cross_entropy_with_logits_v2(
        labels=orig_pred, logits=pred)
    loss2 = tf.nn.softmax_cross_entropy_with_logits_v2(
        labels=target, logits=pred)
    loss_attack = tf.reduce_sum(balance*loss2 + loss1) * weight
    return loss_attack
  • 无目标损失
def untargeted_attack_loss(pred, orig_pred, weight):
    """
    Arguments:
        pred {logits} -- Logits output by threat model (input: adv)
        orig_pred {int} -- Original (correct) prediction by threat model
        weight {float32} -- Attack weight assigned by user (args.attack_weight)
    
    Returns:
        untargeted_attack_loss
    """
    orig_pred = np.eye(1000)[orig_pred]
    loss1 = -tf.nn.softmax_cross_entropy_with_logits_v2(labels=orig_pred, logits=pred)
    loss_attack = loss1
    return loss_attack * weight

2.5平滑损失：Smoothness Loss

使用图像的全变分构造损失函数：

def total_variation_loss(output, weight):
    tv_loss = tf.reduce_sum(
        (output[:, :-1, :-1, :] - output[:, :-1, 1:, :]) * (output[:, :-1, :-1, :] - output[:, :-1, 1:, :]) + \
        (output[:, :-1, :-1, :] - output[:, 1:, :-1, :]) * (output[:, :-1, :-1, :] - output[:, 1:, :-1, :])) / 2.0
    return tv_loss * weight

2.6针对物理条件的图像适应技术

  • 随机选取背景图片，使用 content_mask
  • 缩放、旋转等变换

3.思考

  • 需要手工构造mask，设计伪装效果
  • 缺乏大规模实验，攻击成功率有待进一步实验（使用了150张图片进行实验）
  • 推广到目标检测等模型
  • 进行语义化的尝试
